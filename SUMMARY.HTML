This submission improves the provided GPT baseline while satisfying the assignment constraints: maximum 2000 optimizer steps, fewer than 2 million parameters, CPU-only execution, and training only on the provided corpus.

Architecture

  • GPT-style autoregressive Transformer.
  • Token and positional embeddings.
  • Pre-LayerNorm Transformer blocks.
  • Multi-Head Causal Self-Attention using PyTorch scaled_dot_product_attention.
  • Feed-forward network with SiLU activation.
  • Weight tying between token embeddings and output projection.
