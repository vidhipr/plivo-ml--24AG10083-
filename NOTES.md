# NOTES.md

The final model is based on a GPT-style Transformer optimized within the 2,000-step and 2M parameter constraints. Weight tying was enabled to reduce parameter redundancy and improve language modeling. AdamW with weight decay, linear warmup, and cosine learning-rate decay was used for faster convergence. Gradient clipping was added to stabilize training. The feed-forward network uses the SiLU activation function instead of GELU. Attention projections were modified to remove unnecessary bias terms. The model size was increased while remaining within the allowed parameter budget to better utilize available capacity. Standard GPT-style weight initialization (std = 0.02) was retained for stable optimization. The final configuration was selected based on the lowest development BPB while satisfying all assignment constraints. The focus was on achieving better optimization and parameter efficiency without changing the evaluation interface.
